Colspan in HTML Tables

Colspan is an attribute used in HTML tables to span a cell across multiple columns vertically. It allows you to combine related information under a single cell.

Example: Combining Order Details

Here's a table with a colspan to merge columns for "Order Total" and "Tax":

Name marks
maths 10 10
science 25 10

Benefits of Using Colspan

  • Simplifies table layout for related data.
  • Improves table presentation by grouping similar information.
  • Reduces table size by avoiding unnecessary columns.
