Rowspan in HTML Tables

Rowspan is an attribute used in HTML tables to span a cell across multiple rows.

Example: Merging Header Cells

Here's a table with a header row using rowspan to combine cells for "sem1" and "sem2":

subjects marks
sem1 sem2
science 25 21
maths 25 20

Benefits of Using Rowspan

  • Improves table organization by reducing redundant information.
  • Enhances readability by clearly associating headers with their corresponding data.
  • Saves space by avoiding repeated headers.
